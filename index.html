Web Programlama Sayfası Deneme

Java sadece World Wide Web için sevimli, eğlenceli uygulamaları yazmak için değil. Java genel amaçlı, yüksek düzey programlama dili ve güçlü yazılım platformudur. Java diliyle projeler diğer programlama dillerine göre daha kolay, sağlıklı ve esnek şekilde yapılması mümkün olur. Java ile herşey yapılabilir ve her yerde çalışır, peki bu ne anlama geliyor ? Örneğin Java ile Windowsda yazdığımız kod hiç değiştirmeden Linux'ta aynı şekilde cep telefonunda çalışır.
